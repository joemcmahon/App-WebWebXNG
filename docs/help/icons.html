Overview of icons

General Options

Clicking on Spot the Spider (at the top left-hand corner of each page) takes you back to the front page of the database. Also...

Recent Changes
This icon gives a list of the most recently changed pages, along with the date they were last edited.
Search
This icon takes the user to a search page. This page will normally contain a search form and possibly a list of popular pages.
Edit Email Address
This icon allows you to store and edit your email address in the system. This is used if you request email notification of changes to a page.
Help
This icon takes you to these help pages.
Administration Options
This icon is only displayed to the administrator. It displays a form which allows the administrator to change various options affecting the whole database.
