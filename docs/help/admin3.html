More options

Mail Sensitivity
This number defines the minimum time in minutes between each email notification.
Recent Changes
This number defines the maximum number of pages that are displayed in the recent changes page. If the number is set to zero, the recent changes icon will not appear.
Body Attributes
This string is added to the <body> tag in all of the HTML that is produced.
Menu Background
This defines the background colour of the icon bar. Any standard HTML colour will work.
Highlight Colour
This colour is used to highlight various bits of text throughout the system. It should contrast with the other colours defined above.
Show Cute Icons
This button toggles the appearance of messages from the system. If it is off, then the images of Spot the Spider will not appear.
Show Toolbar as Icons
If checked, this option shows the toolbar as icons (more compact, but more confusing). Unchecking it shows the toolbar as text (faster, but takes more screen real estate).
