WebWeb administration

Options

Admin User
The username of the web administrator. If you change this you will no longer have access to the administration pages, so be careful!
Admin Email
The email address of the administrator. This address is used on pages providing links to "contact the administrator". Administrators must still register their email addresses with the WebWeb server to be able to get update messages.
Default Page
This is the page which is displayed when the WebWeb CGI script is invoked with no arguments.
Search Page
If this page is defined a search icon will appear on the icon bar, leading to a page which can contain a search form and perhaps a list of popular destinations.
System Title
This title is displayed at the top of each page and in any emails that are sent by the system. It can be left blank.
